# Mobile_Data_Display-in-computer
该项目是一款基于 Python 开发的 PC 端工具，核心功能是实现手机 APP 数据在 PC 端的连续采集、识别与展示，聚焦于屏幕指定区域数字数据（如大盘资金净流入）的自动化监控。项目代码下载后，需要点击”手机数据观察程序“，系统自动完成截屏区域配置、数据采集与识别和数据存储与展示功能。我用它来查看并记录同花顺APP中”大盘资金净流入“数据。
